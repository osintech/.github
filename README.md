## ドキュメントについて
共通ドキュメントは `docs/` 配下に集約しています。ディレクトリ構成のイメージ:

```
docs/
└── workflows/        # 共通ワークフローごとのガイド置き場
    └── pr-comment-deploy.md
```

個々のワークフローやその他の詳細は、対応するファイルを参照してください（README に追記する必要はありません）。
